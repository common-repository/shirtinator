== Description ==

Insert your personal Shirtinator Creator wrapped into WordPress without any popup or iframe.

**How it works:**

Please visit [the official website](http://bueltge.de/wp-shirtinator-creator-plugin/510/ "Shirtinator") for further details and the latest information on this plugin.

= Pemises =
* PHP 5
* PEAR-package XML/RPC library
* 720px min. width in the design


== Installation ==
1. Unpack the download-package
1. Upload folder include all files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Got to 'Options' menu submenu 'Shirtinator' and configure the plugin.
1. Write a new page with the content `<!--shirtinator-->` .

See on [the official website](http://bueltge.de/wp-shirtinator-creator-plugin/510/ "Shirtinator").
